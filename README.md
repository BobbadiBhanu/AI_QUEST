# ✨ InspireMe: AI-Powered Motivational Quotes

InspireMe is a dynamic web application that generates motivational quotes with every click. Powered by AI, it serves as a source of daily inspiration — paired with beautiful, calming backgrounds.

## 🎬 Demo

The demo is presented here. You can check out at AI_QUEST/Screen Recording 2026-03-18 234107.mp4

## 🚀 Features

- 🎲 Randomized motivational quotes with every click
- 🎨 Interactive and visually appealing design with smooth transitions
- 🌄 Calming background images for a relaxing experience
